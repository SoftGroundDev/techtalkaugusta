## üìä Converting Markdown to Presentation Formats

### Option 1: Google Slides / PowerPoint
1. Copy each slide section from `2026_slide_deck.md`
2. Create new slide for each `## SLIDE X` section
3. Format with Tech Talk Augusta branding
4. Add images from `/static/carousel-images/` or stock photos

### Option 2: Reveal.js (Web-based)
```bash
# Install reveal-md
npm install -g reveal-md

# Convert to interactive web presentation
reveal-md 2026_slide_deck.md --theme night
```

### Option 3: Marp (Markdown to Slides)
```bash
# Install Marp CLI
npm install -g @marp-team/marp-cli

# Convert to PDF or PowerPoint
marp 2026_slide_deck.md --pdf
marp 2026_slide_deck.md --pptx
```

### Option 4: Slidev (Developer-Friendly)
```bash
# Install Slidev
npm init slidev

# Create slides.md with content from 2026_slide_deck.md
# Run: npm run dev
```

## üé§ Speaking Notes & Tips

### Opening (5 minutes)
- Start with Augusta's potential (Cyber Center, resources)
- Share brief history of Tech Talk Augusta
- Present the bold vision for 2026
- Set tone: "This is OUR community, we're presenting to the board‚Äîyou."

### Body (20 minutes)
- Focus on impact and outcomes, not just activities
- Use specific numbers and targets
- Share the "why" behind each initiative
- Emphasize community involvement and ownership
- Show how pieces connect (hackathons ‚Üí conference ‚Üí growth)

### Growth Agent Section (5 minutes)
- Keep it accessible (not too technical)
- Focus on "what it does" not "how it works"
- Emphasize time savings and better decisions
- Position as enabler, not replacement for humans

### Ask & Close (5 minutes)
- Be specific about what you need
- Make it easy to say "yes" (multiple contribution levels)
- Create urgency (early sponsor benefits, timeline)
- End with inspiring vision of Augusta in 2027

### Q&A (15 minutes)
- Listen fully before answering
- Be honest about challenges
- Connect questions back to impact
- Use questions to reinforce key messages
